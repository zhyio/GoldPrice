# GoldPrice — macOS 悬浮行情桌面小组件

实时显示上海黄金交易所黄金 9999（AU9999，人民币/克）和上证指数，并可展开查看基金持仓，悬浮在桌面最顶层。

## 构建

```bash
cd GoldPrice
./build.sh
```

## 运行

```bash
open GoldPrice.app
```

或直接编译运行（开发模式）：

```bash
swift run
```

## 功能

- 悬浮在所有窗口最顶层
- **金价**: 实时显示 AU9999 人民币/克价格，以及相对上一交易日结算价的涨跌幅
- **上证指数**: 实时显示点位，以及相对上一交易日收盘点位的涨跌幅
- 点击金价或上证数字可在默认浏览器打开对应的东方财富行情页面
- 点击左侧箭头展开基金列表，显示基金名称、持仓金额、持有收益和今日涨跌
- 基金持仓金额和收益按用户提供的持仓截图录入，今日涨跌每小时更新一次
- 中国市场配色：红色表示上涨，绿色表示下跌，灰色表示平盘
- 金价和上证指数每 5 秒自动刷新
- 无 Dock 图标（纯悬浮小组件）
- 可拖动窗口位置
- 深色主题

## 数据来源

| 数据 | API | 用途 |
|------|-----|------|
| 黄金 9999 | [东方财富 AU9999](https://quote.eastmoney.com/globalfuture/AU9999.html) | 当前人民币/克价格、昨结、涨跌额和涨跌幅 |
| 上证指数 | [东方财富](https://quote.eastmoney.com/unify/r/1.000001) | 当前点位、昨收、涨跌额和涨跌幅 |
| 基金估值 | [东方财富基金](https://fund.eastmoney.com/) | 基金盘中估算涨跌幅 |

应用直接采用东方财富返回的涨跌额和涨跌幅。AU9999 以昨结算价为基准，上证指数以昨日收盘点位为基准，与对应东方财富行情页面保持一致。

基金列表中的“金额”和“收益”是持仓快照，不会根据估值自动改写；“今日涨跌”采用东方财富盘中估算涨跌幅，启动时立即请求，之后每小时刷新。数据源未提供估值时显示 `--`，请求失败时保留上一次有效值。
